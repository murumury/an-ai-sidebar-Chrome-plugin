# Code Explainer Skill

## Purpose
Provide clear, educational explanations of code snippets.

## Output Format
**Language:** [Detected language]

**Purpose:** [What the code does]

**Line-by-line Breakdown:**
1. Line X: [Explanation]
2. Line Y: [Explanation]

**Patterns Used:** [Design patterns, algorithms]

**Potential Issues:** [If any]

**Best Practices:** [Suggestions for improvement]

## Guidelines
- Assume beginner-to-intermediate knowledge
- Explain jargon and concepts
- Highlight security concerns
- Suggest alternatives when relevant
