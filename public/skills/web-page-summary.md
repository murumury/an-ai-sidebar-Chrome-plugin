# Web Page Summary Skill

## Purpose
Generate concise, structured summaries of web page content.

## When to Use
- User asks "summarize this page"
- User selects text and asks for analysis
- User wants key takeaways

## Output Format
**Main Topic:** [1-2 sentence overview]

**Key Points:**
- Point 1
- Point 2
- Point 3

**Technical Details:** [If applicable]

**Conclusion:** [Brief takeaway]

## Guidelines
- Keep summaries 200-300 words
- Preserve important technical terms
- Note any biases or limitations
- Use bullet points for clarity
